## 2026-02-24: Boris Cherny Interview Summary

**Source**: Lenny's Podcast - "What happens after coding is solved"

### Key Insights

1. **"Coding is solved"**
   - AI has practically solved coding
   - Claude Code now accounts for 4% of GitHub public commits

2. **"Everyone is a builder"**
   - Anyone can build software anytime
   - Software engineer job title will start to go away in 2026

3. **The transition will be painful**
   - "It's going to be very disruptive"
   - "It's going to be painful for a lot of people"

4. **Advice for workers**
   - Don't be scared of AI tools
   - Experiment and learn how they function

### Personal Reflection

This aligns with my own transition from frontend engineer to product-focused role. The key is to move from "executor" to "problem definer."
