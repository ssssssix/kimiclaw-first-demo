# Boris Cherny 访谈笔记

**来源**: Lenny's Podcast - "What happens after coding is solved"  
**嘉宾**: Boris Cherny（Claude Code 负责人）  
**日期**: 2026年2月19日

---

## 核心观点

### 1. "编程已被解决"
> "AI has practically solved coding"

- Claude Code 占 GitHub 公开提交的 **4%**
- 日活用户上个月翻倍
- Spotify 称最好的开发者从12月起就没写过一行代码

### 2. "未来所有人都是 Builder"
> "Anyone can just build software anytime"

- 产品经理可以直接动手做原型
- 设计师可以用 AI 生成可运行代码
- 创业不再需要技术合伙人

### 3. "软件工程师职位将消失"
> "Software engineer is going to start to go away in 2026"

- 不是工程师会失业
- 而是"软件工程师"这个**职位名称**会消失
- 因为**所有人都能写代码**了

---

## 产品哲学

### Claude Code 成功三原则

1. **不做 IDE，做终端工具** — 融入现有工作流
2. **给团队"无限 token"** — 让团队自由探索 AI 可能性
3. **少做功能，多做深度** — 把核心体验做到极致

### 从 Claude Code 到 Cowork

| 产品 | 定位 |
|------|------|
| Claude Code | 给程序员用的 AI 助手 |
| Cowork | 给所有人用的 AI 同事 |

---

## 类比：印刷术的发明

- **以前**: 只有僧侣能抄写书籍（稀缺技能）
- **印刷术后**: 任何人都能传播思想
- **结果**: 知识爆炸、文艺复兴

**AI 写代码 = 软件开发的印刷术时刻**

## 给打工人的建议

> "Don't be scared of them. Experiment with AI tools and learn how they function."

1. 现在就开始用
2. 理解原理，不只是会用
3. 保持开放，拥抱变化
