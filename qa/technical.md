## OpenAI 模型发展

**Q**: OpenAI 目前有哪些模型？最新的是 GPT-4o 吗？  
**A**:

### 当前主力模型（2026年2月）

| 模型 | 定位 | 说明 |
|------|------|------|
| GPT-5.2 Instant | 主力对话模型 | 最新版，响应更快更自然 |

## AI 行业趋势

**Q**: AI 行业最新动态有哪些？Claude Code 负责人怎么看未来？  
**A**:

### Boris Cherny（Claude Code 负责人）核心观点

1. **"Coding is solved"**
   - AI 已经基本解决了编程问题
   - Claude Code 占 GitHub 公开提交的 4%

2. **"Everyone is a builder"**
   - 未来所有人都能构建软件
   - 软件工程师职位名称将发生变化

3. **AI 投资创新高**
   - 2026年科技巨头 AI 投资约 6500 亿美元
   - 中国模型密集更新（豆包、通义千问、DeepSeek 等）
