### Q1: What are Atlas browser's main features? Is it free? Is it based on Chrome?

**A**:
- **Core Features**:
  - **In-page AI assistance**: Invoke ChatGPT on any webpage without switching tabs, automatically understands page content
  - **Browser memory**: Remembers visited pages and context, cross-session reference (e.g., "find job posts I viewed last week")
  - **Agent mode**: AI can operate browser to complete tasks like search, add to cart, checkout

- **Pricing**:
  - Free: Basic features with usage limits
  - Plus ($20/month): Unlocks Agent mode, more message quota
  - Pro ($200/month): Nearly unlimited, priority access to latest models

- **Engine**: **NOT Chrome**, OpenAI's self-developed browser engine designed for AI integration

- **Availability in China**: Works but requires international network connection

#### Core Insights

**1. "Coding is Solved"**
> "AI has practically solved coding"

> "Anyone can just build software anytime"

**3. "Software Engineer Job Title Will Go Away"**
> "Software engineer is going to start to go away in 2026"

- Not that engineers will be unemployed
- The **job title** "software engineer" will disappear

> "It's going to be very disruptive. It's going to be painful for a lot of people."

- Job descriptions will change rapidly starting 2026

- Some people will be left behind

> "Don't be scared of them. Experiment with AI tools and learn how they function."

- Start using now
- Understand principles, not just usage
- Stay open, embrace change
